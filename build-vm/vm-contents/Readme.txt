VeriWS

------
 
This is a standalone binary for the tool VeriWS developed by Manman Chen and colleagues. 
 

To get started with the tool, follow the steps given below:

1. Open the "PAT 3.exe" (Shortcut on the Desktop).

2. If prompted for the type of project just click Cancel. (Project type selection is not supported yet and would return errors if selected)

 


A usage example is given in the youtube shortcut "How To Use VeriWS on this VM" on the Desktop.
